## Overview

SRF AiOS Schema is licensed under **FRAND/SEP (Fair, Reasonable, and Non-Discriminatory / Standards-Essential Patent)** terms to enable universal adoption while protecting essential governance innovations.

---

## License Model

### **FRAND Principles**

- **Fair:** Licensing terms apply equally to all adopters
- **Reasonable:** No barriers to adoption, royalty-free for conforming implementations
- **Non-Discriminatory:** Available to any person or organization worldwide

### **Standards-Essential Patents (SEP)**

SRF holds patents covering essential mechanisms in cognitive architecture governance:

- **System M.2:** Reflexive Memory Governance and Dual-Bounded Recall
- **System H:** Pre-Execution Ontology Compliance Layer
- **System H.2:** MarginStable Policy Framework
- **System K:** Reflexic Drift Detection for Multi-Agent Systems
- **System A:** Reflex-Governed Mutagenic Execution
- **System M:** Reflexive Governance and Evolution Framework

These patents protect the **governance mechanisms**, not the schema definitions themselves.
